# 0046 以分配与稀缺关系定义资源

- 状态：已接受
- 日期：2026-07-20

## 决策

把**资源（Resource）**定义为：在机制中可被获取、保存、分配、占用、消耗、转换或争夺，并因替代用途与约束形成权衡的规则可追踪存量、容量、权限或机会。

资源是可扩展的专业语义类型，而不是核心语法角色，也不是金币、木材、魔力等固定题材名词。按照具体机制，同一种资源可以由实体、状态、关系、权限、时间窗口或空间机会承担。某个数值或对象只有在参与上述分配、改变可能行动并形成约束时，才在该机制中承担资源角色。

## 影响

分析案例时，应说明某物如何被追踪、取得、持有、分配、占用、消耗、转换或争夺，以及一种用途怎样排除其他用途。不能仅凭名称、重要性或稀有性把对象归为资源。
