# 0069 区分所有权、控制权、持有关系与规则权限

- 状态：已接受
- 日期：2026-07-20

## 决策

正式机制表达不使用无类型的 `Has(participant, object)`，而区分四种第一类关系：**所有权（Ownership）**表示规则认可的持续归属及转让、计分或归还后果；**控制权（Control）**表示谁能为实体选择动作、目标或能力；**持有关系（Custody）**表示对象当前由谁持有或位于谁管理的集合、区域或容器；**规则权限（Rule Permission）**表示谁在指定条件与作用域内获准执行动作、访问信息或调用规则接口。

这些关系可以重合，也可以分别转移、暂停和移除。对象承担资源角色不要求存在私人所有权，空间占据也不自动产生对位置的所有权。

## 影响

卡牌暂时控制、借用物品、共享单位、任务物品、工人放置和公共资源等案例必须说明改变的是哪种关系。自然语言中的“拥有”“持有”“属于”和“可以使用”在进入形式分析前都须消歧，不能据一个关系推断其余权利。
