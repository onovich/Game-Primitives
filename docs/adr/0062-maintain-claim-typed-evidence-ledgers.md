# 0062 维护按结论类型区分的证据账本

- 状态：已接受
- 日期：2026-07-20

## 决策

正式案例为重要结论维护**证据账本（Evidence Ledger）**，记录结论文本与类型、适用的游戏作品／规则集版本／游玩配置、来源及页码／章节／时间码、来源能够支持的范围、已知冲突证据和当前置信度。

规范规则优先由规则文档支持，软件行为由指定版本的可复现实验、录像、源码或技术资料支持，设计意图由设计师演讲、访谈、日志或复盘支持，玩家活动与策略由游玩观察、录像、遥测、行为研究及有边界说明的策略资料支持，体验与认知主张由试玩、访谈、问卷或实验支持，类型与历史身份由研究文献、同期资料、评论和社群或机构用法支持。

一种来源不得越权证明另一类主张：规则允许不等于玩家采用，设计意图不等于体验实现，单次录像不等于普遍行为，攻略存在不等于策略普及。

## 影响

研究记录保留完整账本，网页研究版可为关键结论提供展开链接，成书正文只展示维持论证所需的出处和注释。来源发生冲突时保留冲突与适用范围，不通过删去反例制造确定性。
