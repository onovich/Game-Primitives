# 0081 确定第一轮案例校准名单

- 状态：已接受
- 日期：2026-07-21

## 决策

第一轮四个深度案例为：FIDE 标准国际象棋、锁定配置的无限注德州扑克、Game Boy《俄罗斯方块》A-Type、《农场主》修订版四人基础游戏。

八个标准案例为：Nikoli 标准 9×9 数独、《Dominion》第二版固定王国牌组、《星际争霸：母巢之战》锁定版本与地图的标准 1v1、《Factorio》2.0 基础自由模式、《NetHack》锁定稳定版本的标准单局、《花火》四人五色基础配置、《外交》七人标准地图，以及锁定 IFAB 规则版本的十一人制足球。

## 影响

名单优先覆盖结构差异、跨媒介压力与可核查规则，而非按市场份额或类型标签平均抽样。所有持续更新案例在建档时锁定规则集版本和配置；来源不足时显式保留缺口。平台动作、FPS、格斗、节奏、Minecraft、ARPG、生活模拟、硬核模拟、移动免费游戏和直播运营等明确延后到后续压力测试，不从最终覆盖目标中删除。
