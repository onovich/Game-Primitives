# 0091 不设承诺原语，以决策锁定描述可逆性变化

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-B4

## 决策

本轮不把“承诺”录取为原语。项目用**锁定点**标记动作生命周期中撤回、替换或修正权限消失，或规则成本显著增加的里程碑；用**决策锁定**描述生命周期、状态与关系变化、资源操作、信息公开以及许可和资格变化共同形成的复合结构。

本决定细化 [ADR 0083](0083-model-action-lifecycles-with-game-specific-milestones.md)：其中用于举例的“承诺里程碑”以后按案例改写为具体**锁定点**，动作生命周期本身继续保留。

**决策锁定**是共享分析术语而非独立原语。不同权限可以在不同时点锁定，下注、工人放置和秘密指令也可以具有不同的载体、恢复条件、信息后果与结算资格。

## 影响

机制卡把“成本与承诺”改为“成本与决策锁定”，要求展开锁定发生在哪里、什么仍可修正以及后续可行行动怎样改变。后续契约、秘密同时提交、拍卖、外交与谈判案例若反复暴露无法由现有结构表达的稳定操作，可以重新打开原语候选；本决定不预先否定社会性承诺可能需要额外模型。
