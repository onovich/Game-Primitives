# 0074 把策略建模为跨决策情境的条件性政策

- 状态：已接受
- 日期：2026-07-21

## 决策

定义**策略（Strategy）**为参与者依据目标、观察与信念，在一系列可能决策情境中选择行动的条件性政策。策略不同于一次动作和正式机制；规则提供可行状态变化与反馈，策略说明玩家或代理者怎样在不同条件下选择这些可能性。

不为所有游戏强制规定“战术／战略”的统一时间边界，而记录政策覆盖的实际尺度，例如瞬时、一次交锋、一回合、整局或跨局。套路、技法、开局流程和 meta 按其条件性、尺度、传播范围和证据状态描述；稳定传播但不具规则义务的内容可以成为战略惯例。

## 影响

国际象棋双攻、扑克诈唬、RTS 开局、动作游戏拉扯和社区反制等不能仅因广为使用就列为机制。玩家策略需要由录像、攻略、访谈、赛事或测试证据支持。若 AI 的决策政策由规则明确规定，则可另作为系统过程或规则内容分析，不改变玩家策略的层级身份。
