# 0083 用游戏特定里程碑描述动作生命周期

- 状态：已接受
- 日期：2026-07-21

## 决策

把**动作生命周期（Action Lifecycle）**加入共享工作词汇，用来描述一次输入、动作或自动过程从被规则识别到退出运行边界之间的有序或分支里程碑。案例按需命名识别、承诺、作出、完成、确认、撤回或其他里程碑，不规定所有游戏共享一条固定流水线。

动作生命周期补充而不替代**动作结果**：前者回答“何时走到哪一步”，后者回答输入识别、合法性、结算、实际效果和评价各层面发生了什么。它先作为机制卡可选字段，不直接准入原语词汇。

## 影响

实体国际象棋的触棋、着法 `made`、按钟 `completed` 和裁判确认可以被区分；实时数字游戏也能区分输入采样、候选变化、提交、锁定与后续结算。研究者不得把 FIDE 或任何单款游戏的阶段名推广为普遍语法，也不必为简单动作填满所有阶段。
