# 0089 把资源作为有准入条件的可叠加语义角色

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-B2

## 决策

**资源**不是与实体、状态、关系、容量、权限或机会互斥的对象类型，而是这些结构在特定机制与时间尺度中可以承担的可叠加专业语义角色。承担资源角色不会取消对象的核心语法类型，也不表示它在所有机制中都是资源。

本决定细化 [ADR 0046](0046-define-resources-by-allocation-and-scarcity.md) 与 [ADR 0047](0047-separate-resources-operations-and-economies.md)，不另建资源本体层级。

资源候选必须同时说明：规则追踪的存量、容量、权限或机会；相对于谁和何时发生的稀缺；至少两个竞争用途、竞争者或使用时点；可执行的分配、占用、预留、转移、转换、消耗、释放或刷新等操作；以及当前配置怎样改变未来可行行动。仅仅可计数、重要、稀有或越多越好，不足以通过准入检验。

## 影响

案例模板必须记录载体、稀缺对象、竞争用途、资源操作和时间尺度；关系复杂时再展开容器、所有权、控制权、保管、可用性与索取资格。分数通常仍是评价量，生命值通常仍是状态；只有在具体机制中可被参与者配置到竞争用途时，它们才额外承担资源角色。
