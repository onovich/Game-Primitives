# 0052 把可见性建模为信息访问关系

- 状态：已接受
- 日期：2026-07-20

## 决策

以**信息访问关系（Information Access Relation）**描述谁在何时能够针对哪些规则事实获得何种观察信号。该关系应能记录精度或粒度、延迟、持续性、可靠性和传播范围；公开、私密、隐藏与部分可见是它的不同配置，而不是脱离观察者和时点的对象固有标签。

把查看、揭示、隐藏、共享和广播等视为**信息访问操作**：它们改变访问关系或向参与者生成观察信号。观察信号仍须与世界事实和参与者信念分开；玩家声明可能有来源、误差或欺骗性，不能因此直接写入世界状态。

## 影响

信息机制分析应明确事实、观察者、时点及信号属性。相同事实可以同时对一方公开、对另一方隐藏，并随位置、阶段或动作变化；战争迷雾、私有手牌、反向可见手牌、延迟情报和带噪线索因而能够用同一结构比较。
