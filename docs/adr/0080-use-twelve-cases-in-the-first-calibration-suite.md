# 0080 首轮校准采用四深八标十二案例

- 状态：已接受
- 日期：2026-07-21

## 决策

第一轮**校准样本组（Calibration Suite）**由 4 个深度案例与 8 个标准案例组成。深度案例完成完整证据账本、机制与编排表达、玩家活动证据、边界反例和设计变体；标准案例完成规则范围、核心机制、编排特征、玩家活动及关键模型失效记录。

四个深度案例分别重点施压确定性完全信息、隐藏信息与随机判断、实时操作或物理过程、多机制资源网络与长期反馈。八个标准案例补充逻辑解谜、德式桌游、集合构筑、社会互动、自动化、沙盒和多单位控制等结构。

## 影响

十二案例是便于完成后整体回看的模型校准样本，不是最终类型覆盖清单。首轮完成条件是每个案例能够被当前模型表达或明确记录失败原因、术语跨案例保持可比或具有拆分证据，并形成失败清单与模型版本修订，而不是产出十二篇成稿文章。
