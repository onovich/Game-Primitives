# 0065 区分原语、规则参数、内容数据与运行状态

- 状态：已接受
- 日期：2026-07-20

## 决策

常量、阈值、枚举值、概率分布、空间尺寸、时间长度和内容数值默认作为**规则参数（Rule Parameter）**或**内容数据（Content Data）**，不因频繁出现而成为原语。原语提供有类型的语义操作与组合接口，参数和内容数据配置具体规则表达。

规则参数在明确规则集版本与游玩配置中固定，并在一次游玩期间默认不由该规则系统改变。若某个当前取值能够在运行中被规则读取和修改，则使用既有的状态变量、值和状态事实表达；初始值或基础值仍可以来自参数。内容数据中固定的字段可提供参数，可变字段进入游戏状态。

## 影响

阈值 21、匹配长度 3、8×8 棋盘、5 秒冷却、属性枚举、伤害数值和抽样分布不会分别扩张原语目录。机制家族通过参数化记录变体范围，设计实验可以改变参数并观察结果；只有当改变涉及新的语义操作或组合约束时，才考虑新增或拆分原语。
