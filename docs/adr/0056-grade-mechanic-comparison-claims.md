# 0056 对机制比较结论进行分级

- 状态：已接受
- 日期：2026-07-20

## 决策

机制之间不再只以“相同／不同”描述，而区分四种比较结论：

1. **结构等价**：在声明尺度下重命名实体、题材或符号后，有类型规则槽位、关系、顺序、信息与效果结构相同。
2. **行为等价**：在声明的初始条件、适用范围和观察边界内，产生相同的合法状态变化与可观察结果，即使内部表达不同。
3. **机制家族**：共享明确的结构不变量，同时允许指定槽位、参数或子机制发生有意义变化。
4. **功能类比**：结构不同，但在特定设计情境中承担相似功能。

## 影响

所有“等价”主张必须注明分析尺度、成立前提和观察边界。机制家族条目应列出共同不变量、允许变化的槽位、失去家族身份的边界，以及容易混淆但仅属功能类比的结构。功能相似不能被用作结构相同的证据，行为等价也不能在超出声明条件后继续默认成立。
