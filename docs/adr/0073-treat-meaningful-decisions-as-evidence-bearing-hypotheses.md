# 0073 把有意义决策视为需要证据的假设

- 状态：已接受
- 日期：2026-07-21

## 决策

以**决策情境（Decision Situation）**中性描述某位参与者在特定时间、观察和信念条件下，从当前合法选项集中提交选择并承担其承诺、结算和后果。分析记录决策者、选项集、信息条件、时间与注意力限制、撤销或承诺时点、可能后果与不确定性、反馈时点以及相关目标和评价尺度。

“有意义”不作为原语、机制名称或规则自动保证的属性，而写成**有意义决策假设（Meaningful Decision Hypothesis）**：对于指定玩家水平、信息条件、目标与时间尺度，选项之间存在能够被辨认、学习或表达，并足以影响选择的相关差异。

## 影响

形式上存在多个选项不足以证明有意义决策。研究需检查规则后果是否不同、玩家能否辨认差异、差异是否与其目标相关、选择是否可无成本撤销、何时产生反馈，以及选项是否随知识增长而坍缩为支配解。战略、表达、叙事和即时操作意义可以分别陈述，不能用一个无条件形容词概括。
