# 0075 纳入有证据支持的涌现玩法

- 状态：已接受
- 日期：2026-07-21

## 决策

设计意图不是策略、惯例或玩法模板的准入条件。定义**涌现玩法结构（Emergent Play Structure）**为未被规则直接规定为完整行动序列或目标路径，却由规则、实现行为与玩家策略互动产生，并在游玩证据中反复出现的活动与反馈结构。

按尺度区分单个玩家技法或策略、社群传播的战略惯例，以及由多个机制、玩家活动、反馈和目标形成的涌现玩法模板。每项记录支持它的机制或实现行为、规则集版本和配置、规范规则地位、设计意图证据、社群或赛事裁定、传播范围，以及后来被修复、削弱、禁止或正式吸收的变化。

## 影响

依赖软件漏洞或规范外行为的玩法可以进入研究，但必须同时标明规范规则与实现行为的差异，不能改写成官方机制。一次意外、没有重复活动结构的系统动态或纯粹未预见事件不自动成为涌现玩法；进入玩法模板目录仍须满足编排特征和证据状态要求。
