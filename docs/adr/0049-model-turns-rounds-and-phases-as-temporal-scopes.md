# 0049 把回合、轮次与阶段建模为时间作用域

- 状态：已接受
- 日期：2026-07-20

## 决策

引入**时间作用域（Temporal Scope）**：一段由规则界定进入与退出边界，并在其中规定参与者、行动或响应权限、允许发生的动作、事件与过程以及边界效果的时间组织单位。

“回合”“轮次”“阶段”、波次、昼夜和响应窗口等保留为具体游戏使用的名称，而不假定这些名称具有跨游戏固定的含义或层级。时间作用域可以顺序排列、循环、嵌套、并行或重叠；分析时记录进入条件、有效参与者、权限、内部顺序或并发关系、退出条件以及刷新、结算、回收和后续触发等边界效果。

## 影响

比较不同游戏时，应比较时间作用域的规则结构，而不是直接对齐它们的自然语言名称。此抽象既能描述交替行动，也能描述同时选择、实时运行、事件驱动流程和多尺度循环，不会把“轮次包含回合”强制设为普遍结构。
