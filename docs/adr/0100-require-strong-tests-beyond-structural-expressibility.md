# 0100 在结构可表达性之外要求强检验

- 状态：已接受
- 日期：2026-07-22
- 来源决定：D-C7

## 决策

项目把**结构可表达性**视为理论候选的必要条件，而不是充分条件。一个模型能够为案例填写全部字段，只能说明它暂时没有在该声明范围内失去描述能力；不能据此证明描述无歧义、候选边界稳定、近邻结构可区分、设计后果可预测，或玩家行为与体验主张成立。

第二轮校准采用分层的**强检验**要求：

1. 每个案例比较组必须完成一次**重构测试**；
2. 每组必须在**设计变体测试**与**近邻辨别测试**中至少完成一项；
3. 当研究问题涉及原语准入、拆分、合并、降级或语法槽位边界时，触发**独立标注测试**；
4. 当结论涉及**玩法模板**的实际成立、玩家活动、策略、难度、平衡或体验时，触发**行为取证测试**；没有相应证据就缩小或撤回主张；
5. 第二轮样本组整体采用**反例优先采样**，主动选择当前模型最可能失真的媒介、类型和边界案例。

六种检验组成方法工具箱，但不要求每个案例机械完成全部检验。必做要求落在案例比较组；条件检验只在相应主张出现时触发；反例优先属于整轮采样原则。

## 依据

第一轮十二案证明当前模型能够在声明范围内描述多种规则结构，也暴露了三个限制：同一描述可能容许多种重构；同名结构可能被错误合并、异名结构可能被错误分开；规则材料只能证明某些玩家活动可能发生，不能证明其实际频率、难度或体验效果。

若继续只以“能够写进模板”为通过标准，项目可以通过增加字段和术语无限吸收反例，最终失去可证伪性。重构、区分、预测、独立标注和行为材料分别针对这些失败方式，并允许候选退回、降级或被拒绝。

## 影响

- [第一轮案例校准](../../research/calibration-cycle-1.md)在“结构采样与门审完成”的限定意义上关闭；行为、复现和测量缺口继续保留在[校准证据积压](../../research/calibration-evidence-backlog.md)。
- [第二轮校准协议](../../research/calibration-cycle-2-protocol.md)成为后续组样和执行强检验的依据；协议获准不等于样本组已经确定或第二轮已经启动。
- 案例研究包模板升级为 v0.5，增加组级强检验记录与条件触发检查。
- **暂准入**原语候选只有通过相应强检验，才能获得更高置信度；结构可表达不能单独把候选升级为稳定结论。
- 项目仍处于阶段一·开放研究；关闭一轮校准不触发书稿阶段或项目阶段转换。
