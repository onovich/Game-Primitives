# 0047 区分资源、资源操作与经济系统

- 状态：已接受
- 日期：2026-07-20

## 决策

分别记录三种分析对象：**资源**是被规则追踪、分配并形成权衡的存量、容量、权限或机会；**资源操作**是产生、获取、储存、分配、占用、释放、转移、交换、转换、消耗或刷新资源的规则处理；**经济系统**是这些资源及其来源、存量、容量、所有权、流动、去向、时机与反馈构成的持续运行网络。

资源操作可以表现为动作、结算或效果，不因涉及资源就自动成为独立机制。经济系统既不要求出现货币或商店题材，也不要求存在多种资源，但至少需要资源状态与操作在时间中形成可辨认的流动关系；单个数值或一次支付不足以构成经济系统。

## 影响

资源案例优先使用“来源—存量—分配／转移／转换—去向”的流动图，并补充容量、所有权、可见性、时机与反馈。以上是机制内部及机制之间的分析词汇，不在“原语—机制—玩法模板—游戏”的公开主链中增加新层级。
