# 0077 把平衡与公平写成有类型主张

- 状态：已接受
- 日期：2026-07-21

## 决策

“平衡”与“公平”不是原语、机制固有属性或无需解释的设计目标。使用**平衡主张（Balance Claim）**分别描述结果、阵营／角色、选项、策略生态、经济或进程关系；使用**公平主张（Fairness Claim）**分别描述程序、机会、信息、匹配、分配或玩家感知。

每项主张必须注明比较对象、目标指标、玩家群体或能力区间、规则集版本与游玩配置、时间尺度、可接受误差和行为或体验证据。不同维度可以互相冲突，不用一个总分掩盖差异。

## 影响

对称规则不自动证明结果平衡，胜率接近不自动证明策略多样，随机程序一致不自动证明玩家感知公平，补偿机制改善机会也可能改变程序公平评价。案例和设计练习必须先命名准备调整的平衡或公平维度，再选择参数、实验和指标。
