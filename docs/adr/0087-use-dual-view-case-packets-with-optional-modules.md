# 0087 案例包采用双视图与可选专业模块

- 状态：已接受
- 日期：2026-07-21

## 决策

案例研究包升级为 v0.2，同时维护可追溯的**教学最小视图**与**研究充分视图**。前者刚好足以解释当前机制与设计问题；后者足以复核合法性、边界、裁定或实现行为。前者必须指出省略内容并能定位到后者，两者不得形成互相冲突的事实版本。

模板为机制卡增加动作生命周期、调度语义、执行来源、裁定权和实体身份效果；为范围卡增加关键制品、完整性标识和复现状态；为证据账本增加规则实现层。资源、集合、规则修改器、状态附着等专业模块改为按需展开。

## 影响

研究包继续保存足够严谨的材料，但不要求读者入口先承受实现级状态和所有可选字段。网页未来可以折叠或切换视图；书稿仍将依据教学目标重新写作，而不是直接把研究包排版出版。
