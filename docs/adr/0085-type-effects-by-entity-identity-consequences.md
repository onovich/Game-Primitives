# 0085 按实体身份后果标记效果

- 状态：已接受
- 日期：2026-07-21

## 决策

把**实体身份效果（Entity-Identity Effect）**加入共享工作词汇，作为**效果**的子类型。案例按需标记效果对实体存在、类型、组成与连续追踪关系的影响：保留、创建、移除、替换、重定型、拆分或合并。普通属性修改默认保留身份。

这些操作不是七个新的顶层原语。身份判定依据规则能否持续单独指称和追踪对象，而非程序内存地址、画面相似或题材名称。

## 影响

王车易位可表达为两个保留身份实体的原子联合更新；升变必须声明是重定型还是替换；Game Boy《俄罗斯方块》的锁定可以表达为活动复合实体的移除或拆分，并把四格物化为固定占据。后续合成、召唤、变身、分裂与单位编组案例将继续压力测试这组操作。
