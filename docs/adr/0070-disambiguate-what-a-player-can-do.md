# 0070 消歧玩家“可以”做什么

- 状态：已接受
- 日期：2026-07-20

## 决策

正式分析不使用未消歧的“玩家可以做 X”，而区分：玩家在物理或技术上能否提交相应输入；是否具有该类动作的规则权限；具体动作在当前状态是否满足全部条件而合法；规则是否允许发起但不保证成功的尝试；某种结果在随机、对抗与状态约束下是否可达；以及动作在玩家策略判断中是否值得选择。

定义**动作合法性（Action Legality）**为：在给定世界状态、时间作用域与规则集下，一个具体规则动作是否满足适用权限、前置条件、成本、目标和约束的判定结果。合法性不保证结算成功或产生预期效果，也不保证策略价值。

## 影响

案例应分别记录输入被系统识别、动作合法性检查、尝试结算和效果结果。界面的启用／禁用状态可以反馈或执行合法性，但显示错误不自动改变规范规则；程序若作为执行来源阻止动作，则同时记录其实现行为与规范来源。玩家物理上能够违反规则不构成规则权限。
