# 0082 成对研究并在校准门集中修订

- 状态：已接受
- 日期：2026-07-21

## 决策

第一轮案例校准先建立案例研究包模板 v0.1，再按以下顺序执行：国际象棋与 Game Boy《俄罗斯方块》A-Type；无限注德州扑克与《农场主》修订版；数独与花火；Dominion 与 Factorio；星际争霸与外交；NetHack 与足球。

第一对深度案例后进入校准门 A，重点检查规则语法、时间模型和案例字段；第二对深度案例后进入校准门 B，重点检查信息、资源、编排和玩家活动；八个标准案例完成后进行首轮总体汇总。

## 影响

案例研究期间发现的问题先进入失败清单，共享术语与模型优先在校准门集中修订，避免十二个案例继承不断漂移的模型版本。只有阻塞后续表达的严重错误允许提前修订，并须记录原因。校准门检查跨案例语义一致性、字段负担、证据缺口、玩家活动推测、教学可读性以及术语拆分、合并或降级需求。
