# 0090 记录带时间范围的信息访问与观察后效

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-B3

## 决策

继续使用**世界状态—观察—信念**三层模型，不增加“信息生命周期”或第四种信息状态。**信息访问关系**必须带观察者与时间范围；相关案例通过可选的**观察后效**记录一次观察之后的保留期限、复查条件、验证方式、记录能力、传播范围与可能衰减。

本决定细化 [ADR 0041](0041-separate-world-observation-and-belief.md) 与 [ADR 0052](0052-model-visibility-as-an-information-access-relation.md)，不改变其三层边界。

规则保证、实现或物质载体支持、游玩规范允许，以及玩家实际上记住、相信、记录或传播的内容必须分别陈述。

## 影响

案例模板的信息表增加组合字段，而不是增加新的公开层级或强制所有案例填写多张表。记忆游戏、侦查、合作沟通、日志、回放和观战等案例可以紧凑比较“看过一次”“持续可见”“可复查”和“可验证”等不同结构，同时不把规则许可写成玩家认知事实。
