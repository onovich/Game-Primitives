# 0086 在证据账本中追踪规则实现层

- 状态：已接受
- 日期：2026-07-21

## 决策

证据账本为规则相关主张增加**规则实现层（Rule Realization Layer）**标记，区分规范文本规定、软件或物质制品实装、裁判／程序／参与者执行，以及具体游玩实例中被观察到的实际发生。

同一主张可以跨多层一致，也可以显式冲突。该标记属于研究证据方法，不是游戏本体的新层级，不插入“原语 → 机制 → 玩法模板 → 游戏”。设计意图、玩家行为与体验仍使用现有主张类型记录。

## 影响

规则书、反汇编、裁判实践和实际对局不再被当作互相可替代的证据。案例应说明来源能直接支持哪一层、不能越权支持哪一层，并把未完成复现保留为缺口。
