# 0093 分离结构校准与行为证据状态

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-B6

## 决策

案例门审分别记录**结构校准状态**与**行为证据状态**。结构校准可以在声明范围内通过，而行为、策略与体验证据继续待补；结构通过只表示模型能够忠实表达案例规则结构、边界与已发现的失败，不证明理论完备，也不证明预期玩家活动已经发生。

本决定扩展 [ADR 0059](0059-track-evidence-status-for-gameplay-patterns.md)、[ADR 0062](0062-maintain-claim-typed-evidence-ledgers.md) 与 [ADR 0086](0086-track-rule-realization-layers-in-evidence-ledgers.md)，但不把证据状态插入游戏本体主线。

研究主张继续区分规则事实、结构推论、个案行为观察、跨案例支持和原型验证。只有提出行为、策略或体验主张时，才要求相应经验材料；纯结构案例可以明确记录“行为未主张”。所有门审状态均可被后续反例重新打开。

## 影响

Gate B 记为“结构校准通过；行为证据待补；理论结论仍可修订”。项目可以继续进入标准案例而不等待对称完整的玩家观察，但德州扑克与《农场主》的行为缺口继续保留在失败清单和后续取证计划中。规则许可、结构激励和玩家实际行为不得因门审通过而混写。
