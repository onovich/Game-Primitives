# 0064 以语义与设计检验准入原语

- 状态：已接受
- 日期：2026-07-20

## 决策

原语不追求跨所有目的绝对不可再分，也不默认采用代码实现的最小函数作为边界。候选原语以声明分析尺度下的语义不可约性、组合接口和设计变异价值为主要准入标准。

**原语准入检验（Primitive Admission Test）**要求候选：承担单一规则语义职责；具有明确类型和输入／输出或作用接口；移除或替换后会改变合法行动、状态、信息、顺序或结算；能够跨案例复用；不能被现有原语无损替代；能够说明可填规则槽位和组合约束；并能通过替换、删除或修改提出可测试的设计预测。

实现细节只有在改变可能行动、信息、时机、裁定或执行时才进入规则语义。只有单个案例支持的新概念可以保留为候选，不为追求词汇整齐立即接受或删除。

## 影响

“战斗”“Roguelike”“牌库”和无类型的“随机”等复合、类型或配置名称不能直接作为原语。每个准入条目仍须声明语义角色、类型、尺度、暂不继续拆分的理由、组合接口、支持案例、反例和研究状态，并随案例压力测试接受拆分、合并或废弃。
