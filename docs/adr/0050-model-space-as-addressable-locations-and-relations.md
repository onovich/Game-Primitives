# 0050 以可寻址位置及其关系建模空间

- 状态：已接受
- 日期：2026-07-20

## 决策

把**空间**视为专业语义家族，并以**规则空间（Rule Space）**描述由规则能够单独寻址的位置及其可查询或可改变关系构成的结构。网格、图、区域、轨道、连续坐标和功能槽位是不同的**空间模型**，而不是互不相关的题材原语。

位置可以是棋盘格、节点、地块、轨道位置、连续坐标或非地理性的行动槽位。邻接、连接、包含、方向、距离、容量、占据和可达性等关系决定空间的规则含义。视觉地图、棋盘或场景只有在影响合法行动、状态、查询或结算时才进入机制语义；实体布局若直接执行这些约束，则同时是规则结构。

## 影响

空间案例应分别记录位置域、关系或几何、容量与占据、寻址方式以及呈现映射，不能只写“使用地图／网格”。没有视觉场景的文字游戏和抽象桌游也可以具有规则空间，具有空间画面的游戏则未必让画面中的全部位置关系参与机制。
