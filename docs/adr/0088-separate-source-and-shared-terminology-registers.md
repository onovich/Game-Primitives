# 0088 分离来源语域与共享术语

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-B1

## 决策

来源术语只在四类位置保留原有词义：带定位的**原文引文**、忠实但不冒充逐字引文的**规则转述**、术语映射，以及讨论词语本身的元语言段落。前两类合称**来源语域**。首次出现重要词面碰撞时立即建立映射；完成映射后，项目自己的分析行文使用**共享术语**。

术语映射不强迫一对一翻译，必须允许等价、来源较窄、来源较宽、部分重叠、拆分为多个共享概念和无对应项。来源术语与共享术语各自具有稳定概念身份；网页与目录以后可用独立内部 ID 保存身份，翻译只改变显示名称。

## 影响

共享术语继续以加粗和网页链接呈现；来源术语保留来源形式，并可用斜体、代码样式、引号或来源限定辅助识别。样式、字体、颜色和语言差异都不是唯一判据。案例模板增加按需术语映射；规则转述必须明确是转述，不能使用引文样式制造逐字引用的印象。
