# 保持四阶段公开主线，并把严谨性放入三个箭头

项目继续以“原语 ──语法化──> 机制 ──编排──> 玩法模板 ──实例化──> 游戏”作为面向读者的核心模型，不为容纳运行时动态、体验、媒介或情境而增加新的公开层级。该模型是一把可以双向使用的设计缩放尺：向下用于拆解，向上用于组合与检查；它不是要求设计者依次完成四个阶段的生产流水线，实际创作可以从体验意图、既有机制、材料限制或具体游戏进入并反复往返。

研究后台仍检查触发、条件、结算、状态变化、反馈、运行时动态、界面、媒介与社会情境，但把这些作为定义字段、证据链和压力测试维度，而不是主线中的新节点。首部成书方案默认完整展示四阶段链路，使原语研究能够抵达可玩的具体结果，同时把主要解释深度放在原语、语法和机制；是否最终分册，等待完整案例模块与目标读者测试后再决定。
