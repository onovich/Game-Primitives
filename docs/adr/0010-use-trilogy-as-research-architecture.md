# 以三个箭头组织长期研究，但不预先承诺三册出版

项目暂以《从原语到机制》《从机制到玩法》《从玩法到游戏》对应语法构成、运行编排和具体实例化三种不同问题，作为《游戏原语》的长期研究架构。它不是已经确定的出版合同或硬性分册边界：只有当每一部分都证明拥有独立问题、方法、读者成果和足够材料时，才最终决定是否出版三卷；无论是否分卷，每一部分都必须适度越过自己的主箭头，完成从分析到可玩结果的小型闭环。

第一部分的渐进教学目标是把资深玩家训练为能够拆解、表达、修改、创造并测试机制的人，结业成果暂定为一个由核心机制主导、规则闭合、经过外部试玩与迭代的微型游戏。后续部分分别提高到持续玩法原型和具有具体作品身份的小型游戏，以避免第一部分退化为原语词典，也避免三部分成为互相割裂的瀑布流程。
