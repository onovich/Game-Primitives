# 0098 使用时间、不确定性与测量剖面

- 状态：已接受
- 日期：2026-07-21
- 来源决定：D-C5

## 决策

案例分析按需使用**时间剖面**、**不确定性剖面**与**测量剖面**。三者都是固定比较问题的研究结构，复用现有时间、调度、随机、信息、参数、观察与证据概念；不录取为新**原语**或公开层级。

**时间剖面**先记录时间通道或作用域的所有者与驱动来源、输入与响应窗口、承诺／截止／公开、更新与读取快照、暂停／积压／恢复／补偿、终止及测量方式，再允许用“回合”“实时”“同时”等熟悉标签作摘要。同一案例可以有多只时钟、计数器和重叠作用域。

**不确定性剖面**以特定参与者、对象和时段为边界，记录不确定性的来源类型、生成者或选择者、样本空间或约束、观察与揭示、可控性、可复现性及受影响结果。规则概率抽样、他人选择、隐藏既成状态、连续物理或环境扰动、执行误差、系统复杂性与实现未知不得统称为同一种“随机”。

**测量剖面**围绕一个待比较量，分开规则或实现参数、正式裁定值、实际观察、派生指标、玩家诊断与研究评价，并记录定义或公式、单位、时间窗、数据来源或仪器、适用案例单位、误差和证据边界。

## 依据

国际象棋、Game Boy《俄罗斯方块》《星际争霸》《外交》、NetHack 与足球表明，“回合”“实时”“同时”和比赛时长会遮蔽不同的输入、更新、锁定、暂停和终止结构。德州扑克、《Dominion》、NetHack 与足球又表明，概率抽样与不可预测性不是同义词。

《Factorio》的标称速率、实测流量、端到端吞吐、瓶颈诊断和优化评价，以及足球的规则比赛时长、裁判补时、外部计时观察与比赛评价，证明数值身份也必须分开。

## 影响

案例研究包模板 v0.4 用**时间剖面**替代单一“基础时间模型”选项，强化**不确定性剖面**，并在证据约定中增加**测量剖面**。自然语言标签仍可用于教学摘要，但不能单独支撑结构比较。

三张剖面均按需填写。简单案例不必制造多只时钟、概率模型或性能指标；省略时说明它们与当前研究问题无关即可。
