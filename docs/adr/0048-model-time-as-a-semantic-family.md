# 0048 把时间建模为专业语义家族

- 状态：已接受
- 日期：2026-07-20

## 决策

不把“时间”当作一个内容单一的原语，而把它作为专业语义家族，分析其中的先后与同时关系、行动调度、持续时间、响应窗口、冷却与延迟、截止条件以及从瞬时更新到跨局进程的嵌套尺度。

“回合制”“实时制”“阶段制”和“事件驱动”首先描述规则如何组织动作、事件、过程与效果，并不自动构成独立机制。同一游戏可以同时使用多种时间结构。只有当某段时间或行动机会能够被追踪、分配、占用、消耗或争夺并形成替代用途时，才把它进一步标记为**时间资源**。

## 影响

机制分析在时间差异会改变合法行动、可观察中间状态或参与者机会时，应记录顺序、调度、持续与延迟、窗口以及所在尺度。不能用一个笼统的“回合制／实时制”标签替代这些规则，也不能把所有经过的时间都解释为资源。
