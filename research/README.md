# 研究记录

这里保存文献条目、阅读笔记、案例证据、术语沿革、反例和开放问题。书稿可以追求流畅，研究记录必须保留可追溯性。

## 内容标记

建议在笔记中显式区分：

- **[来源]** 可由书籍、论文、演讲、访谈或游戏材料支持的陈述。
- **[项目定义]** 为本项目建立的操作性定义。
- **[假设]** 尚待案例或形式化检验的判断。
- **[观察]** 对具体游戏或设计现象的记录。
- **[反例]** 对当前分类或论证构成挑战的材料。
- **[问题]** 暂不作答、需要后续研究的问题。
- **[决定]** 已说明理由、可在未来修订的编辑或建模选择。

## 来源最低信息

尽可能记录作者、标题、年份、载体、版本、页码或章节、稳定链接与访问日期。引用游戏时记录平台与所分析版本；规则随版本变化时，不把当前状态写成永久事实。
